# 🚀 Smart Inventory & Sales Analytics dApp

## 📖 Description

This project is a decentralized Inventory and Sales Analytics application built on the Stellar blockchain using Soroban smart contracts.

It enables users to manage inventory items and perform sales transactions while automatically tracking revenue — all stored securely and transparently on-chain.

---

## 🔥 Key Features

* 📦 Add and manage inventory items
* 📋 View all items
* ❌ Delete items
* 💰 Sell items (transaction system)
* 🔄 Automatic stock reduction
* 🧾 Sales history with timestamp
* 📊 Total revenue calculation
* 🔐 Fully decentralized

---

## 💡 Innovation

This project is more than a basic CRUD application.
It simulates a **Web3 Point of Sale (POS) system** where:

* Inventory is stored on blockchain
* Transactions are recorded immutably
* Revenue is calculated automatically

---
